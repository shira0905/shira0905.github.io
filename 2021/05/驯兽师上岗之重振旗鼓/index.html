驯兽师上岗之重振旗鼓

Posted on 2021-05-04 Edited on 2021-05-05 In uncategorized

许久不见啊~ 是喜爱的五月了,一切都应该明朗起来才对.

近期状态

五一5天假期算是很清静地休息了一阵, 虽然我也知道学校的假期和公司的假期应该取交集而非并集, 但是用5天时间clean up和restart一下也不过分吧! 本文主要是整理一系列改革措施.

自从离开香港之后, 虽然说在公司作息会正常一些, 很少出现工作学习时候的心流状态了, 分析了一下, 有一下原因:

  1. 开放式的工作环境, 很必须时刻关注的社交软件
  2. 饮食比之前更复杂了, 至少两餐还会有零食, 同时导致体重失控
  3. 在学校的科研压力导致无法在工作时候能量条拉满

心流状态的消失, 驯兽师也像是出离岗位了一样, 伴随着大脑中野兽的肆虐横行, 比如暴饮暴食, 无节制地购物, 工作学习低效, 阅读时间减少等等等等. 所以啊, 要采取些措施了!

基本欲望满足

  • 节食不太行, 按往期的经验来看, 容易暴饮暴食, 而且一旦吃超出计划的食物会让自己有很强的挫败感

  • 还是生酮吧, 不会饿肚子, 还能大吃一餐, 吃的都算健康. 不生酮总会忍不住吃零食.

优化状态的活动

  • 进食仪式感: 进食洁癖消毒, 以及进食前食物摆放的仪式感
  • 工作学习仪式感: 桌面上垃圾扔掉, 咖啡加奶, 手机放在抽屉, 键盘桌面耳机笔消毒
  • 中午吃饭又很破坏状态, 还是尽量外出运动, 健康餐, 看书, 回家睡觉.
  • 听歌, 巴洛克歌单, 蹦迪歌单, 慢跑歌单

野兽冷静期

暴饮暴食时策略

  • 收藏的视频 “暴饮暴食”
  • 看书
  • 去16楼
  • 去健身房拉伸
  • 在家可以洗澡躺床上

淘宝美团点评策略

  • “瘦了一切完美”
  • 删掉购物软件, 让朋友代买

甜蜜协商

自我对话

  • 集中注意力
  • 上一步进行到哪里了?
  • 接下来30分钟完成xx

编故事

  • 等到完成列表上面的事情才可以吃东西
  • 再坚持工作5分钟
  • 再忍不吃饼干10分钟, 并且去看点视频吧

自我人设认知:

  • 立志保持25岁面孔身体的程序媛

  • 工作思考不含糊的独立工作狂

  • 热爱学习尝试新事物的好气宝宝

  • 跑马拉松不腿软 运动能力不错的活力girl

  • 崇尚婴儿态养生自然, 不争不抢不自恃的道家粉

  • 疯狂消毒的洁癖处女座

  • 胃容量有限的精致美食家

  • 工作时巴洛克抖腿 追随简约经典棉麻材质主义的古典粉

反馈表

回报感觉计划

基本设定表 长期

  1. 目标
  2. 重要度检查
  3. 具象想象
  4. 反向规划
  5. 每日任务设定

实践设定表 每天的

  1. 每日任务选择

    类似每日的plan不宜太多

  2. 对比目标与障碍

  3. 克服障碍的对策

  4. 提问: 将1中的任务转化成提问格式: [圆头] 在[时间] 在[地点] 执行[任务] 了吗?

  5. 现实想象: 尽可能真实地描绘出完成任务的过程

    类似之前八点跑步时候的绿纸条流程

  6. 固定式视觉提示: 能把联想到完成任务相关的事物放在醒目的地方

    类似自制的达摩克里斯之剑

2,3,4是没有尝试过得

比如, 任务是把代码run起来.

障碍: 上厕所时候被手机吸引住
策略: 上厕所就打开微信阅读, 再不行小红书也可

障碍: 上厕所或者倒水杯零食箱吸引住
策略: 除了早上的催屎咖啡, 不要喝水, 减少上厕所的频次

速效简易版

  1. 积极想象: 眼前任务完成后会有什么积极事情发生, 写下来
  2. 积极选择: 挑选1中最积极的事情, 打星
  3. 对比目标与障碍
  4. 校级选择: 挑选3中最难解决的问题
  5. 提问: 将1中的任务转化成提问格式: [圆头] 在[时间] 在[地点] 执行[任务] 了吗?

情绪

这个技巧的重点是一边想象着“如果情绪是某种物质”，一边像科学家一样观察自己的情绪。想象不出来的时候，请尝试问自己以下问题。

  • 这种情绪是什么颜色的？
  • 这种情绪的大小是什么样的？黄豆大小？网球大小？还是高楼大小？
  • 这种情绪位于自己身体的哪个部位？占了多少空间？
  • 如果触碰这种情绪，会有什么样的感觉？是硬还是柔软？是粗糙还是光滑？
  • 这种情绪有温度吗？是热？是冷？还是常温？
  • 这种情绪有什么动作吗？是在振动？静止？还是在跳动？如果在动，速度大约是多少？

坚持以上训练的时间越长，你对野兽的反应的理解就越深刻。“工作稍微有点多，野兽就会想上网”“不管做什么工作，一个小时左右的时候野兽就会食欲大增”，像这样，你会知道在什么情况下野兽容易变得注意力不集中。但是，事先做好思想准备，驯兽师就很难被冲动所挟裹。今后如果再出现类似的情况，请把它当作锻炼驯兽师的机会。与情绪保持距离是提高专注力的有效技巧。想看SNS的时候，工作进展不顺、想摆脱焦虑的时候，只要暂时与分散你注意力的情绪保持距离，你就能迅速回到原来的工作中。但是，请不要在这种时候刻意控制自己的情绪。
