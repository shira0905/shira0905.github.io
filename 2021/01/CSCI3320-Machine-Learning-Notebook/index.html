covariance matrix

PCA is the Eigen decomposition on the covariance matrix

实现

Step 1

Centralize the data by subtracting the mean vector from each data point.

In python, for a matrix X, X.mean(0) is the average of each row vector and X.mean(1) is the average of each column vector.

You can use numpy.ones(N, 1) to create a N×1 column vector.

Estimate the covariance matrix.

You can compute the variance using the numpy.cov(…).

Step 2

Perform the Eigen decomposition on the estimated covariance matrix S.

S is a PSD matrix. So its eigenvalues are non-negative. If you get a very small negative number in the computation, this is due to the float error. Just ignore them.

you can use numpy.linalg.eigh(S) to perform the Eigen decomposition for the PSD matrix
