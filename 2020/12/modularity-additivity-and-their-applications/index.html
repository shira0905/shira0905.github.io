Modularity, additivity and their applications *

Posted on 2020-12-21 Edited on 2021-01-14 In Research , basic

近似算法个人最常用的理论依据。有空多学学组合优化还是挺有趣的。还有些【坑】未填，后续补上。

关于submodular

从subadditive讲起

Definition 5: A set function $f$ is subadditive if for all $A\cap B= \emptyset$ with $A,B \subseteq N$ the value of the union does not exceed the sum of the indicidual values $f(A)+f(B) \geq f(A\cup B)$.

【仔细品品最后一句话】

定义

$f:2^N \to \mathbb{R}$ is submodular 定义不想打了。

平时熟悉的定义是Lemma 8。Lemma 8 等价于 Definition 6，【有空把左推右 右推左的证明写一下】。

性质

  • 【证明submodular函数都是subadditive】

    【反过来如果一个函数是subadditive的，应该没法推出submodular的吧？】

    【additive是不是还要求A和B不重合？好像重合也不影响证明？】

  • submodular函数的求和还是submodular函数

  • submodular函数的加权求和（权重非负）也还是submodular函数

贪婪算法

有的时候贪婪算法就可以得到目标的最优解，比如说最小生成树算法。一般来讲判断是否能够达到最优值可以用Matroid这个东西【待填坑】。如果不能那么最常用的就是用 Submodularity + Monotonicity 推导出贪婪算法的结果 achieves a $(1- 1/e = 63\%)$-approximation of the optimum。

假设再给 $f$ Monotonicity 这个性质，你的任务是找到一个大小为k的子集S使得f达到最大，那么不同大小的k会造成什么影响呢？该理论的一个漂亮的地方是，如果你函数有Submodularity和Monotonicity这两个性质那么至少63%的效果是可以保证的。当然在实际中，常常可以高于这个值。

证明该bound

submodular函数都是subadditive

定义$f_S$

对于$S$ 的marginal contribution 定义一个关于$S$ 的函数 $f_S(T) = f(S\cup T)-f(S)$。

分析$f_S$

根据submodular的性质，显然submodular等价于【省略不想打了】。“事实上这里定义的marginal contribution可以看做是在离散数据上微分的推广，这里测量的正是一种增量。”【这句话值得品一品。】

以下三个命题是等价的:

  1. $f$ is submodular
  2. $f_S$ is submodular
  3. $f_S$ is subadditive

证明思路，先用 1证2，再2证3（证明submodular函数都是subadditive），再3证1。
